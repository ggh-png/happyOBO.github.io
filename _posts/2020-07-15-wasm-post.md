# 1. Ttypes

## 1.1 Value types

value types classify the indiviual values that WebAssembly code can compute with and the values that variable accepts.
$$
valtype ::= i32 |i64 |f32 |f64
$$

## 1.2 Result Types

$$
resulttype ::= [vec(val)]
$$

## 1.3 Function Types

$$
functype ::= resulttype \rightarrow resulttype
$$
